Overview of MCP Servers

Helping non-engineers understand what MCPs are and how they are used.

Author: Kyle McMeekin Date: Jun 25, 2025

What is MCP?

Before we get into what an MCP (Model Context Protocol) server is, let’s talk about standards. Think of HIPAA in healthcare or NIST in various industries. These standards ensure everyone follows the same rules, which opens the door for scaling and innovation.

MCP is a new set of rules and standards for how an AI brain, like a Large Language Model (LLM), talks to different apps and online tools. It's a technology standard like REST APIs or HTTPS, but designed specifically for AI integration.

Who Developed MCPs?

The concept of MCPs was introduced by Anthropic in November 2024 to standardize the way AI models integrate with external tools and data sources. Since then, hundreds of companies, including Stripe, have developed their own MCP Servers, and the standard is gaining significant popularity.

The Problem: Why We Need a Translator

An AI by itself is like a brain that's great at predicting words but can't actually do anything in the real world. It can't send an email, check your calendar, or search the web on its own. To perform actions or access specific, real-time data (like your company's sales figures), it needs to connect to external tools.

The problem is, every tool speaks a different "language." Connecting an AI to a calendar, then to email, then to a search engine becomes an engineering nightmare, as you have to build a custom integration for every single tool.

The Solution: How MCP Fixes This

MCP solves this by creating a unified layer, or one shared language, for all tools. Instead of the AI learning dozens of different languages, it just speaks the one MCP language. The MCP system acts as the master translator that communicates with all the individual tools, bridging the gap between understanding and execution.

The Four Parts of the System

1. The Client: The app you use (e.g., Anthropic, Cursor, Gemini).
2. The MCP Protocol: The rulebook for the shared language.
3. The MCP Server: The master translator managing conversations (e.g., the Stripe MCP Server).
4. The Service: The actual tool that performs the job (e.g., the Stripe platform).

Stripe’s MCP Server In Action

As of June 2025, Stripe’s MCP Server has several capabilities. As these are extended, AI tools like Cursor can "understand" and, more importantly, "DO" Stripe tasks directly from their interface. The Stripe MCP handles the abstraction of these tasks, like creating customers and payment links.

What This Means for the Future

For anyone interested in building with AI, this is a game-changer. Think of MCPs as creating standard Lego pieces. Once these pieces are available, it will be much easier for developers to snap together different tools and services to build powerful and genuinely helpful AI assistants.
